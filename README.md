# Kupola Landing

Статический сайт-каталог для минаретов, куполов, полумесяцев и комплектующих.

## Структура

- `index.html` — главная страница каталога.
- `*-minaret.html`, `crescent-*.html` — страницы товаров.
- `assets/landing.css` — стили.
- `assets/landing.js` — интерактивность каталога, галерей и форм.
- `assets/*.png` — изображения товаров.
